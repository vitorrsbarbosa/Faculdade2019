CircleCI:
    -Tem uma grande variabilidade de configurações para automatizar o pipeline desde o commit até o deploy, integrado com versionadores de código, roda automaticamente testes e builds num container ou máquina virtual permitindo testar cada parte separadamente para garantir qualidade mínima
    -É gratúito utilizando apenas um container, indo até $3150/mês utilizando 64 containers com até 16 aplicações diferentes concorrentes paralelas
    -https://circleci.com
